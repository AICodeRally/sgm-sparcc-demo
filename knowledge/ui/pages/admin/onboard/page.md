# Admin / Client Onboarding

## Purpose
Step-by-step wizard to onboard new clients, creating their tenant and initial configuration.

## When to use
Use this page when bringing a new client organization onto the platform.

## How to use
1. Start the onboarding wizard
2. Enter client organization details (name, slug, industry)
3. Configure initial settings and preferences
4. Review and confirm the setup
5. Complete onboarding to create the tenant

## Key data
Organization name, slug, industry, contact information, initial configuration.
