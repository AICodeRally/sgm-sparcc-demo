# Admin / User Management

## Purpose
Create, view, and manage user accounts and their role assignments.

## When to use
Use this page to add new users, modify roles, or review user access.

## How to use
1. View the list of users with their roles and tenant assignments
2. Search or filter users by name, email, or role
3. Click on a user to view details or modify their role
4. Use the Add User button to create new user accounts

## Key data
User name, email, role (SUPER_ADMIN, ADMIN, MANAGER, USER, VIEWER), tenant assignment.
