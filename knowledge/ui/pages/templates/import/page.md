# Templates / Bulk Import

## Purpose
Bulk import plan templates from CSV files for efficient template management.

## When to use
Use this page when migrating templates from another system or adding multiple templates at once.

## How to use
1. Download the CSV template to see the required format
2. Prepare your CSV file with template data
3. Drag and drop or select your CSV file to upload
4. Review the validation results and fix any errors
5. Preview the templates to be imported
6. Click Import to create the templates

## Key data
Template code (TPL-XXX-000 format), name, description, plan type, category, status.
