# Settings / AI Features

## Purpose
Enable or disable AI-powered features including the AskSGM assistant and other AI capabilities.

## When to use
Use this page to configure which AI features are available to users.

## How to use
1. Review available AI features and their descriptions
2. Toggle features on or off based on requirements
3. Configure API keys if using external AI services
4. Save changes to apply the new configuration

## Key data
Feature toggles, API configuration, usage limits.
